Interacciones elegantes sin perder performance

Fondo interactivo con ondas al clic y partículas que reaccionan. Todo en canvas, respetando prefers-reduced-motion.

Probar interacciones Ver detalles Volver a FigiKhan
  • Canvas 2D optimizado (desynchronized, DPR capped)
  • Ondas con mezcla aditiva de color
  • Partículas con enlaces dinámicos

Características clave

Ondas al clic

Generá ondas que se expanden y repelen/atraen partículas con una curva suave.

Enlaces dinámicos

Líneas entre partículas cercanas con alpha proporcional a la distancia.

Modos de fondo

Cambiá entre Ondas y Partículas desde el botón del header.

Accesible

Si el usuario reduce movimiento, el fondo se vuelve estático y liviano.

Rendimiento

requestAnimationFrame, listeners passive, y IntersectionObserver.

Personalizable

Editá variables CSS para paleta, radios, sombras y densidad.

Detalles nativos

¿Qué hace el modo “Ondas”?
Crea círculos concéntricos que se expanden; las partículas cercanas reaccionan con fuerzas físicas suaves.

Preguntas frecuentes

¿Funciona en móviles?

Sí. Los toques también generan ondas. El consumo está limitado para no agotar batería.
